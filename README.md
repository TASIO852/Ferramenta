# Ferramenta multi-funções

Ferramenta criada com multiplas funçoes para facilitar coisas simples com excel e arquivos de diversos formatos

> Funcionalidades

- Conversor XML
- Conversor PDF
- Diagrama generator
- Envio de email
- Junção de pastas de excel e pastas csv
- Extrator de PDF
- .......

> Libs Principais

- PysimpleGUI
- diagrams
- PyPDF2
